# Web Library Management System

## 📌 نظرة عامة عن المشروع
نظام إدارة مكتبة متكامل يهدف إلى تنظيم العمليات المكتبية وتسهيل استعارة وإرجاع الكتب[span_0](start_span)[span_0](end_span). تم تطوير هذا المشروع كبيئة عمل تعاونية منظمة تقسم المهام بوضوح بين خمسة مطورين لضمان بناء نظام قوي وقابل للتوسع[span_1](start_span)[span_1](end_span). 

## 🚀 التقنيات المستخدمة
* **لغة البرمجة الأساسية:** PHP[span_2](start_span)[span_2](end_span)
* **قاعدة البيانات:** MySQL[span_3](start_span)[span_3](end_span)
* **الواجهات الأمامية:** تقنيات الويب القياسية مع تصميم متجاوب (HTML, CSS, JavaScript)[span_4](start_span)[span_4](end_span)

## ✨ المميزات الرئيسية
* **نظام استعارة ذكي (Circulation Logic):** يتضمن تطبيق قواعد إعارة صارمة مثل الحد الأقصى للاستعارة (3 كتب للمستخدم)، التحقق من توفر الكمية، ومنع استعارة نفس الكتاب مرتين في وقت واحد[span_5](start_span)[span_5](end_span).
* **إدارة محتوى متكاملة (CRUD):** تتيح للمسؤولين إضافة، تعديل، وحذف الكتب، بالإضافة إلى رفع صور الأغلفة وإدارة تفاصيل كل كتاب[span_6](start_span)[span_6](end_span).
* **لوحة تحكم إدارية:** نظام متكامل لإدارة المستخدمين، متابعة أنشطة النظام، وإنشاء تقارير وتصديرها بصيغة CSV[span_7](start_span)[span_7](end_span).
* **أمان وموثوقية عالية:** 
  * حماية النظام من هجمات XSS باستخدام دوال تطهير المخرجات `htmlspecialchars`[span_8](start_span)[span_8](end_span).
  * منع حقن قواعد البيانات (SQL Injection) عبر تطهير المدخلات باستخدام `intval`[span_9](start_span)[span_9](end_span).
  * استخدام المعاملات المتزامنة (Transactions) في قاعدة البيانات (Commit/Rollback) لضمان سلامة العمليات أثناء إرجاع الكتب وتحديث كمياتها[span_10](start_span)[span_10](end_span).
* **واجهة مستخدم تفاعلية (UX/UI):** تشمل صفحة رئيسية ديناميكية، نظام بحث متقدم، وصفحة شخصية (إعاراتي) لفصل الكتب المستعارة حالياً عن سجل الإعارات السابقة، مع حساب تلقائي لمواعيد الإرجاع المتوقعة[span_11](start_span)[span_11](end_span).

## 📁 الهيكلية وتقسيم المهام
تم بناء المشروع وفق هيكلية تنظيمية تضمن جودة العمل، حيث تم تقسيم المجلدات إلى مجلد للدمج (`Integration`) ومجلدات للعمل الفردي (`Team`)[span_12](start_span)[span_12](end_span). شملت أدوار الفريق:
1. **مهندس النظام وقائد الفريق:** إعداد البنية التحتية، نظام المصادقة (التسجيل والدخول)، إدارة الجلسات، ودمج الملفات[span_13](start_span)[span_13](end_span).
2. **مدير المحتوى:** معالجة بيانات الكتب وعرض التفاصيل[span_14](start_span)[span_14](end_span).
3. **مطور الواجهات:** تصميم الصفحات وتطوير نظام البحث[span_15](start_span)[span_15](end_span).
4. **مطور منطق الإعارة:** برمجة العمليات المعقدة للاستعارة والإرجاع[span_16](start_span)[span_16](end_span).
5. **مطور الإدارة:** برمجة لوحة التحكم الإدارية واستخراج التقارير[span_17](start_span)[span_17](end_span).

## ⚙️ التثبيت والتشغيل
1. قم بتحميل أو استنساخ المستودع (`git clone`).
2. قم باستيراد ملف قاعدة البيانات `dbco.sql` (الموجود في مجلد `Backups`) إلى خادم MySQL المحلي لديك لإنشاء الجداول والبيانات الأولية[span_18](start_span)[span_18](end_span).
3. تأكد من إعداد بيانات الاتصال بقاعدة البيانات بشكل صحيح داخل ملف `includes/db.php`[span_19](start_span)[span_19](end_span).
4. النسخة الجاهزة والنهائية للتشغيل موجودة داخل مجلد `Integration`[span_20](start_span)[span_20](end_span).
